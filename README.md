# Sorting Algorithms Benchmark & Testing

> **Note:** This is a personal learning project focused on Java software architecture, test-driven development (TDD) with JUnit 5, and performance benchmarking (Big O notation).

## About the Project
A scalable framework for implementing, testing, and benchmarking sorting algorithms in Java. Key focus areas include:
* **Clean Architecture:** Interface-driven design (`SortingAlgorithm`) with modular utility classes.
* **Unit Testing:** Comprehensive edge-case testing (e.g., empty arrays, sorted arrays, duplicates) using JUnit 5.
* **Benchmarking:** Automated dataset generation and CSV export for time complexity analysis.

## Tech Stack
* **Java** (JDK 20+)
* **Gradle** (Kotlin DSL)
* **JUnit 5**
